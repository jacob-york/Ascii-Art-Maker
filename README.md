# Ascii-Art-Studio

Desktop Application for creating Ascii Art from various sources.

Alpha release includes (I'm working on some minor GUI adjustments before I do so):
  -A Graphical User Interface written in JavaFX.
  -Support for (fast) rendering of PNGs and JPGs into Ascii Art, with adjustable resolution and the option to invert shading.
  
Feature Complete (beta) will include:
  -Full support for generating images from:
      -desktop screenshots.
      -external camera feed (i.e. webcam).
  -Full support for rendering animated Ascii Art from:
      -Video files.
      -Live input stream (real-time rendering) like a webcam.
